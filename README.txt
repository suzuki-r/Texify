## What’s this?

This is an updated Javascript file for spires.app, available at
  http://member.ipmu.jp/yuji.tachikawa/spires/
for Mac OS X.


## What’s new?

Minor fixes of \langle, \rangle, \varphi, \varkappa, \vert


## How to replace

- Open Applications in Finder
- Right-click spires.app
- Choose Show Package Contents
- Open Contents/Resources
- Replace tex.js
